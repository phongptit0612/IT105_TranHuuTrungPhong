Bảng So Sánh ERD và Class Diagram

| Tiêu chí   | ERD                             | Class Diagram                         |
| ---------- | ------------------------------- | ------------------------------------- |
| Mục đích   | Mô tả dữ liệu & quan hệ         | Mô tả lớp, thuộc tính, phương thức    |
| Thành phần | Entity, Attribute, Relationship | Class, Attribute, Method              |
| Quan hệ    | 1–1, 1–N, N–N                   | Association, Aggregation, Inheritance |
| Giai đoạn  | Phân tích & thiết kế DB         | Phân tích & thiết kế hệ thống OO      |
| Ví dụ      | ERD: Sách–Tác giả               | Class: Book có method getTotal        |
